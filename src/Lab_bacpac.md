Экспорт и импорт баз данных: Способ переносить нужную БД с одного сервера на другой
---

Способ, который работает в условиях ограниченных прав учётки

### Экспорт

1. Кликаем правой кнопкой на нужную базу данных -> Задачи -> Экспорт приложения уровня данных
<img src="assets/lab_bacpac/IMG_9461.JPG" alt="IMG_9461" width="700"/>

2. Жмём "Далее"
<img src="assets/lab_bacpac/IMG_9462.JPG" alt="IMG_9462" width="700"/>

3. Выбираем место, куда сохраним файл с БД и даём ей имя после нажатия на кнопку "Обзор".
Она должна сохраниться с расширением *.bacpac
<img src="assets/lab_bacpac/IMG_9463.JPG" alt="IMG_9463" width="700"/>

4. Жмём "Готово"
<img src="assets/lab_bacpac/IMG_9464.JPG" alt="IMG_9464" width="700"/>

5. Ждём
<img src="assets/lab_bacpac/IMG_9465.JPG" alt="IMG_9465" width="700"/>

6. Экспорт готов
<img src="assets/lab_bacpac/IMG_9466.JPG" alt="IMG_9466" width="700"/>

7. Полученный файл выглядит примерно вот так:
<img src="assets/lab_bacpac/IMG_9467.JPG" alt="IMG_9467" width="350"/>

### Импорт

1. Кликаем правой кнопкой на папку "Базы данных" -> Импорт приложения уровня данных
<img src="assets/lab_bacpac/IMG_9468.JPG" alt="IMG_9468" width="400"/>

2. Жмём "Далее"
<img src="assets/lab_bacpac/IMG_9469.JPG" alt="IMG_9469" width="700"/>

3. Жмём "Обзор" и выбираем файл *.bacpac
<img src="assets/lab_bacpac/IMG_9470.JPG" alt="IMG_9470" width="700"/>

4. Указываем название для новой импортируемой БД. Жмём "Далее"
<img src="assets/lab_bacpac/IMG_9471.JPG" alt="IMG_9471" width="700"/>

5. Жмём "Готово"
<img src="assets/lab_bacpac/IMG_9472.JPG" alt="IMG_9472" width="700"/>

6. Ждём
<img src="assets/lab_bacpac/IMG_9473.JPG" alt="IMG_9473" width="700"/>

7. Импорт готов
<img src="assets/lab_bacpac/IMG_9474.JPG" alt="IMG_9474" width="700"/>
